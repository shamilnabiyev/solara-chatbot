# Solara Chatbot 

Simple Solara Chatbot using local Ollama

<div align="center">
    <img src="docs/imgs/chatbot-gui.png" alt="Chatbot GUI" width="500"/>
</div>


1. Install dependencies

    ```bash
    pip install -r requirements
    ```
2. Run the solara app

    ```bash
    solara run chatbot.py 
    ```
